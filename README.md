## 🤹‍♂️ 개발 환경 및 기술 스택

| 항목 | 내용 |
|---|---|
| **프로그래밍 언어** | Java 11 |
| **프레임워크** | ✨**Spring Boot** 2.7.18 |
| **빌드 도구** | Gradle 8.8 |
| **데이터베이스(DB)** | PostgreSQL 16.3 (AWS RDS) |
| **ORM** | Spring Data JPA 2.2 |
| **프론트엔드(View)** | ✨**React** 18.3.1 |
| **개발 도구(IDE)** | IntelliJ IDEA, VSCode |
